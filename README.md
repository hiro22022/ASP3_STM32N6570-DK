# TOPPERS/ASP3 on STM32N6570-DK

## ベース

 TOPPERS/ASP3 on STM32H747 Discovery をベースに STM32N6570-DK への TOPPERS/ASP3 移植を行います。

 AIの支援を受けています。というより、ほぼ AI です。

## CMake 対応について

 実験的です。

 というか、Cursor のエージェントウィンドウで、方針を相談したつもりだったが、どうやらエディターウィンドウで編集中のこちらに実装までしてくれた。

 使い方のわかっていない AI には注意が必要ですね。

## ライセンス

TOPPERS/ASP3 本体は TOPPERS ライセンスです。
STM32CubeMX で生成したコードについては、そちらのライセンスに従います。
